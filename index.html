Multi-Computer Data Sharing

⚠️ How this app stores data (important to understand)

This app saves data in your browser's localStorage — storage built into each individual browser on each individual computer. That means changes made on one computer do not automatically appear on another. Below are your options to share data across computers, from simplest to most powerful.

Your Options

📋 Manual Export / Import Easiest

Use the Export Data button in the Staff tab to download a .json file. Copy that file (email, USB, shared drive) to the other computer and use Import Data to load it. Best for one person making all schedule changes.

☁️ Host on a Shared Drive Easy

Save the HTML file to a shared Google Drive or OneDrive folder. All computers open the same file. Each computer still has its own localStorage, so you still need to export/import after major roster changes — but everyone sees the same base file.

🌐 Google Sheets Backend Recommended

Host the HTML file anywhere and use Google Apps Script as a free backend. Staff data and PTO requests live in a Google Sheet. Any computer that opens the app reads from and writes to the same sheet in real-time. Requires ~1 hour of setup.

🖥️ Local Network Server Technical

Run a simple Node.js or Python server on one computer on your school's WiFi network. All other computers on the same network connect to it by IP address. Full real-time sync, no internet required. Requires IT setup.

🌐 Recommended: Google Sheets Setup (step-by-step)

  1. Go to script.google.com and create a new project
  2. Paste the Apps Script code below into the editor
  3. Deploy → New Deployment → Web App → "Anyone" access → Copy the URL
  4. Replace YOUR_SCRIPT_URL in this HTML file's <script> block
  5. All computers that open the same HTML file will share live data
// Google Apps Script (paste into script.google.com) const SHEET_ID = 'YOUR_GOOGLE_SHEET_ID'; function doPost(e) { const data = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName('data'); sheet.getRange('A1').setValue(JSON.stringify(data)); return ContentService.createTextOutput('ok'); } function doGet() { const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName('data'); const val = sheet.getRange('A1').getValue(); return ContentService.createTextOutput(val || '{}') .setMimeType(ContentService.MimeType.JSON); }
